## Modifications

StoryForge AI includes the following substantial changes from the original:

- Complete rebrand from AIStoryWriter to StoryForge AI
- New `storyforge/` package with reorganized architecture
- Character Builder, World Builder, and Story Timeline features
- PDF and EPUB export capabilities
- Story History with persistent index
- Streamlit web dashboard with modern UI
- Improved prompt system for chapter continuity and consistency
- New CLI entry point (`forge.py`) with extended feature flags
